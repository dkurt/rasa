## OpenVINO backend

Intel OpenVINO is a library which brings optimized deep learning proimitives for IA.

## Config options

There are multiple config options related to OpenVINO backend:

* `use_openvino` - `True` to enable OpenVINO backend. `False` to use default networks backend (TensorFlow)
* `openvino_max_length` - an integer value which specifies fixed input sequence lenght. Extra input values will be zero-padded with no changes for the output. Use this option to achieve the best OpenVINO efficiency.
